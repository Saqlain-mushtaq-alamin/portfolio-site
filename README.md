# Md Al Amin — Developer Portfolio

🌐 **Live Site:** [https://saqlain-mushtaq-alamin.github.io/portfolio-site/](https://saqlain-mushtaq-alamin.github.io/portfolio-site/)

A modern, professional portfolio website showcasing my projects, skills, and services as a Full Stack Developer and CS student.

## ✨ Features

- **Modern Design** — Clean, dark-themed UI with smooth animations
- **Fully Responsive** — Works beautifully on all screen sizes
- **Single Page** — Smooth scroll navigation between sections
- **Interactive** — Typewriter effect, project filters, AOS animations
- **Contact Form** — Built-in contact form for reaching out
- **Fast Loading** — No heavy frameworks, pure HTML/CSS/JS

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- AOS (Animate on Scroll)
- Devicon (Skill icons)
- Google Fonts (Inter, JetBrains Mono)

## 🚀 Deployment

This site is deployed using **GitHub Pages**. To deploy your own:

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under "Source", select **Deploy from a branch**
4. Choose **main** branch and **/ (root)** folder
5. Click **Save**
6. Your site will be live at `https://YOUR-USERNAME.github.io/portfolio-site/`
